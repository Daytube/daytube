여러 플랫폼 사용과 페이지 이동의 불편함

사랑하는 연인과의 데이트를 위해 유튜브에 검색해본 홍대 데이트.. 하지만 우리는 유튜브에 나온 장소 정보를 얻기위해 보던 영상을 멈추고 포털사이트에 또 검색을 했어야 했습니다. 위치정보를 얻기위해 지도에 검색을 하고, 평점을 보고, 리뷰를 일일이 확인하고..

데이트 영상과 장소들을 한눈에

특정 키워드로 검색된 유튜브 영상들과 해당 영상에 나오는 장소들을 한눈에 확인 할 수 있고 원하는 영상은 따로 페이지 이동없이 시청가능합니다. 이를 통해 키워드로 검색된 영상들과 장소들을 데이트 정보 얻기라는 목적에 맞게 유튜브 인터페이스 보다 더 편리하게 확인할 수 있습니다.

해당 장소의 정보를 얻기위해 여러 매체를 이용할 필요가 없습니다.

특정 장소 태그를 클릭하면 해당 장소에 대한 주소, 평점, 영업시간, 연락처, 위치 정보를 얻을 수 있습니다. 또한 리뷰를 5가지 카테고리 (맛, 가성비, 분위기, 서비스, 재방문의사)로 감성분석한 수치를 확인 할 수 있습니다.

How to develop DayTube 데이튜브

실제 유튜브 영상 스크래핑
특정 검색어로 검색된 유튜브 영상의 제목과 더보기란의 text를 스크랩핑합니다 .
데이터 전처리
추출된 text에서 이모티콘, 특수문자, 절대 상호명이 될 수 없는 불용어를 제거합니다.
정규식 패턴 적용
유튜버들이 일반적으로 상호명을 적는 방식들을 정규식으로 패턴화하여 상호명이 추출될 수 있도록 등록합니다.
형태소 분석
KAIST SWRC에서 개발한 Hannanum(한나눔) 형태소 분석기를 통해 명사 태그를 가진 단어들을 추출합니다.
지도Api
카카오맵, 구글맵을 사용하여 해당 상호를 검색하고 상호에 대한 장소가 있다면 그 장소에 대한 정보와 리뷰 데이터를 추출합니다.
감성 분석
고객의 리뷰 내에 숨겨진 잠재적인 주제를 발견하기 위해 LDA 모델을 사용하여 분위기, 가성비, 맛, 재방문의사, 서비스 토픽 별로 감성분석
